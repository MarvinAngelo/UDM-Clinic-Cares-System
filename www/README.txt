UDM Clinic Cares System - Database Import Instructions

To import the latest database, please follow these steps:

1. First, login to your admin account:
   - Username: admin
   - Password: 1234567

2. Navigate to "Manage Staff" section

3. In the "Database Backup & Restore" container:
   - Click the "Browse" button
   - Navigate to either:
     * C:\UDM Clinic Cares System\clinic_data.sql
     OR
     * Your Google Drive folder > ClinicCaresBackup folder > select the latest database file

4. Select the database file and click the "Import" button

5. Your latest database will now be uploaded and ready to use

Note: Make sure you have a backup of your current database before importing a new one.
